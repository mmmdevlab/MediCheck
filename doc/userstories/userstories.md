# User Stories

### Patient Stories (Core User)\*\*

**Authentication:**

- As a patient, I want to create an account so I can start tracking my healthcare
- As a patient, I want to sign in securely so my health data is protected
- As a patient, I want to sign out so my data stays private on shared devices

**Appointment Management:**

- As a patient, I want to add a new appointment so I can keep track of my doctor visits
- As a patient, I want to see all my upcoming appointments so I don't forget them
- As a patient, I want to view past appointments so I can reference my health history
- As a patient, I want to edit appointment details so I can update times or add notes
- As a patient, I want to delete appointments so I can remove cancelled visits
- As a patient, I want to mark appointments as completed so I can track my attendance

**Health Status Tracking:**

- As a patient, I want to log how I'm feeling daily so my caregivers know when to check in
- As a patient, I want to see my feeling history so I can track my health patterns
- As a patient, I want caregivers to see my status without revealing medical details

**Task Management:**

- As a patient, I want to create action items after appointments so I remember what the doctor told me to do
- As a patient, I want to check off completed tasks so I can track my progress
- As a patient, I want to see all pending tasks so I know what needs attention

**Caregiver Coordination:**

- As a patient, I want to grant caregiver access to my schedule so trusted people can help me
- As a patient, I want to revoke caregiver access so I control who sees my information
- As a patient, I want to request help for specific appointments so caregivers know when I need support
- As a patient, I want to see which requests were accepted so I know help is coming

### Caregiver Stories

**Dashboard & Monitoring:**

- As a caregiver, I want to see all my assigned patients so I know who I'm helping
- As a caregiver, I want to view each patient's upcoming appointments so I can plan support
- As a caregiver, I want to see patient's latest health status so I know when to check in
- As a caregiver, I want to see patient's open tasks so I can offer assistance

**Support Request Flow:**

- As a caregiver, I want to receive support requests so I know when patients need help
- As a caregiver, I want to accept or decline requests so I can manage my availability
- As a caregiver, I want to see request details (date, time, type) so I can prepare
- As a caregiver, I want to mark requests as completed so patients know I fulfilled my commitment

**Privacy Boundaries:**

- As a caregiver, I want to see appointments without medical details so I respect patient privacy
- As a caregiver, I want read-only access so I can't accidentally modify patient data
